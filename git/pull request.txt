pull request하는 방법

1. 해당 git hub에 들어가 fork를 누른다.(나의 리파지토리로 복사하는것)
2. 나의 리파지토리에 해당 프로젝트가 있으면 이를 클론하여 프로젝트를 툴에 띄운다.
3. 해당 프로젝트의 이슈를 확인하고 이를 위한 코드 분석과 코드리뷰를 받기위한 pull request 신청폼을 확인한다.
4. 문제 코드를 수정하고 나의 브랜치를 파서 해당 브랜치에 git push까지 하고
5. 나의 git hub에서 pull request를 하면 된다. (이 때 앞서 pull request 신청폼을 이슈에서 다른 사람들은 어떻게 했는지 확인하고 보내면 더 좋다.)
